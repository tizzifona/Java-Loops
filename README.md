# Java Multiplication Table

## ✅Overview

A console application that, given an integer n, prints its multiplication table from 1 to 10. Each multiple n * i (where 1 <= i <= 10) is printed on a new line in the format: n x i = result.


## ✅Example

n = 5

Output:<br>
5 x 1 = 5<br>
5 x 2 = 10<br>
5 x 3 = 15<br>
5 x 4 = 20<br>
5 x 5 = 25<br>
5 x 6 = 30<br>
5 x 7 = 35<br>
5 x 8 = 40<br>
5 x 9 = 45<br>
5 x 10 = 50<br>
